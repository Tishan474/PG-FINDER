# PG-FINDER
A web application to help users find Paying Guest (PG) accommodations easily.

## Features
- Search PG accommodations by location
- Filter by price, amenities, and availability
- View detailed listings with photos
- Contact landlords directly

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Python (update as needed)
- Database: MongoDB / MySQL (update as needed)
